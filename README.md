# 🇨🇳 푸통푸통(PutongPutong) 중국어 단어 학습기

본 프로젝트는 중국어 학습자의 학습 효율성을 극대화하기 위해 개발된 웹 기반 인터랙티브 학습 도구입니다.

## 🛠️ 제작 및 기술 개발
- **기술 스택**: HTML5, CSS3, JavaScript (Web Speech API 활용)

## 📚 원작 교재 정보
- **교재 저자**: **HwajinJIN** & 중국어 연구자 1인 공동 집필
- **콘텐츠 특징**: 저자들의 독자적인 교수 설계와 연구 데이터를 바탕으로 구성된 비영리 교육 콘텐츠입니다.

## 🎯 프로젝트 목적
- **비영리 및 무료 제공**: 학습자들이 경제적 부담 없이 고품질의 학습 도구를 이용할 수 있도록 지원합니다.
- **학습 효율 제고**: 시각적 단어 카드와 청각적 발음 피드백을 결합하여 학습 효과를 높입니다.


## ✨ 주요 기능
1. **단어 카드**: 1~12과 핵심 어휘 학습
2. **필기 연습**: 한자 직접 쓰기 및 획순 확인
3. **발음 연습**: AI 음성 인식을 통한 실시간 발음 교정

## ⚠️ 발음 인식 관련 유의사항
- **인식 기술**: 본 도구는 브라우저 내장 Web Speech API를 활용합니다.
- **정확도 안내**: AI 음성 인식의 특성상 100% 완벽한 판정을 보장하지 않습니다. 주변 소음, 마이크 성능, 통신 환경에 따라 인식률에 차이가 발생할 수 있습니다.
- **학습 권장**: 본 기능은 정답 여부를 가리는 절대적 기준이 아닌, 학습자의 발음을 스스로 모니터링하기 위한 **'보조적 참고 자료'**로 활용해 주시기 바랍니다.

### 🌐 이용 환경 안내
- **플랫폼 제한**: 본 도구의 발음 듣기 및 음성 인식 기능은 **웹(Web) 브라우저 기반**으로 작동합니다.
- **최적 브라우저**: 모바일 앱이나 일부 모바일 브라우저에서는 기능이 제한될 수 있으므로, **데스크톱 환경의 크롬(Chrome) 브라우저** 사용을 권장합니다.
- **모바일 이용**: 스마트폰 환경에서는 기기 및 브라우저 설정에 따라 발음 재생이 원활하지 않을 수 있음을 알려드립니다.
